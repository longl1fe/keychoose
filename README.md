# KeyChoose

KeyChoose — минималистичный фоновый анализатор клавиатуры для Windows и Linux. Программа собирает статистику нажатий клавиш и по итогам выдает рекомендации по оптимальному размеру клавиатуры: 40%, 60%, TKL или полноразмерная.

Программа предназначена для кратковременного запуска на фоне (например, несколько дней), после чего можно получить сводку использования клавиш. Все данные хранятся локально в SQLite, никакая личная информация не собирается.

---

## Возможности

- Фоновый сбор нажатий клавиш и сохранение в SQLite базу  
- Подсчет частоты использования каждой клавиши  
- Рекомендации по размеру клавиатуры на основе активности  
- Простое меню при запуске с выбором: начать запись, сбросить базу с подтверждением, выйти  
- Безопасный выход из записи с комбинацией клавиш `Ctrl + Alt + Q`  
- Локальное хранение данных, никаких данных о паролях или личной информации
